# **写出斐波那契函数 F(m) 的递归和非递归算法**

```
F(m)
├── 0                     m=0
├── 1                     m=1
└── F(m-1)+F(m-2)         m>0
```

```c
int recur(int m){
    if (m == 0 || m == 1)
        return m;
    else
        return recur(m-1) + recur(m-2);
} //时空复杂度分别为 O(2^N) 和 O(N)


int nonrecur(int m){
    if (m == 0 || m == 1)
        return 1;
    int dp[m+1];
    dp[0] = 1;
    dp[1] = 1;
    for (int i = 2; i <= m; i++)
        dp[i] = dp[i-1] + dp[i-2];
    return dp[m];
} //时空复杂度分别为 O(N) 和 O(N)
//可以输入 m=60 比较一下两者的速度
```
